Detailed Description

Abstract extension Please extend one of the base classes.

Note
All the extensions derived from any of the base class should be unique and definitive. Any conflicting name loaded on server may crash the server. e.g, a PostArchiveExtension may be extended to a class called SendToS3BucketExtension
