Real-time centralized logging

Residue is an effort to standardize application logging and making logging as a microservice by implementing simple logging API, easily configurable, build-in log rotators, simple way to format your log messages according to need of your application and most of all language-independent.

Fast, minimal and single-header cryptography library

Mine is fast, single-header minimal cryptography implementation for small-medium projects that cannot afford to link to external libraries.


Cryptography library for C++

Ripe is a cryptography library and CLI tool. Essentially, a wrapper library for third-party cryptography library.
