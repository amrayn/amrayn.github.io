Detailed Description

Provides AES crypto functionalities.

This is validated against NIST test data and all the corresponding tests under test/ directory are from NIST themselves.

Please make sure to use public functions and do not use private functions especially in production as you may end up using them incorrectly. However the source code for AES class is heavily commented for verification on implementation.
